# Power Plant

#### A fidgetech project

#### By Jason Falk

## Technologies Used

* _Html_
* _Javascript_
* _Git/Git Bash_

## Description

_A game where you can feed, water, and photosynthesize plants. One plant is also a wizard and can cast spells._

## Setup/Installation Requirements

* _In Git Bash type git clone https://github.com/JasoFal/PowerPlant.solution.git_
* _Install all packages with <mark>$ npm install<mark>_
* _Build project using webpack with <mark>$ npm run build<mark>_
* _Start dev server with <mark>$ npm run start<mark>_

* _Linting JS files in the <mark>src<mark> folder with <mark>$ npm run lint<mark>_
* _Run tests with Jest using <mark>$ npm run test<mark>_
